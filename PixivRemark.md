## Pixiv 小说标记
### 标记符原文

| **标记符** | **功能** |
| :---- | :---- |
| [newpage] | 分页 |
| [chapter:本章标题] | 分章 |
| [pixivimage:作品ID] | 添加插画 |
| [jump:链接目标的页面编号] | 内部跳转页码 |
| [[jumpuri:标题 > 链接目标的URL]] | 超链接 |
| [[rb:汉字 > 假名]] | 注音 |
| **汉字《假名》*** | **注音** |


**遇到《书名号》的时候，P站会问你要不要置换成P站的注音标记符号  
很多作者会误选置换，造成排版上的错误**  
置换前：**汉字《假名》**  
置换后：**[[rb:汉字 > 假名]]**
